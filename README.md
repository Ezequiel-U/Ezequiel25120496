# Pokémon y sus Tipos – Práctica 2

## Descripción
Página web estática creada con HTML, CSS y JavaScript.
Utiliza Bootstrap para el diseño y la PokéAPI para mostrar Pokémon reales.

## Ejecución local
Abrir el archivo `index.html` en cualquier navegador web.

## Publicación
La página fue publicada mediante GitHub Pages:

https://TU_USUARIO.github.io/TU_REPOSITORIO/

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap
- PokéAPI
